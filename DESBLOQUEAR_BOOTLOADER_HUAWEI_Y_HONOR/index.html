DESBLOQUEAR BOOTLOADER HUAWEI Y HONOR

Juanro @Juanro49
Juanro @Juanro49
HUAWEI

¿Es posible desbloquear el bootloader?

Por todos es sabido que el día 24 de Julio de 2018 Huawei decidió dejar de proporcionar el código de desbloqueo de bootloader huawei de todos sus dispositivos, por lo que si no lo has obtenido antes de esa fecha no podrás desbloquear el bootloader de tu dispositivo Huawei o Honor, al menos gratuitamente.

Esta decisión ha sido muy discutida por muchos usuarios de la comunidad de Android, ya que este movimiento perjudica a la libertad de este sistema operativo, su mayor atractivo, no permitiendo hacer root ni instalar custom roms en dispositivos de la marca china.

Por suerte, gracias a HWdog y los clientes que hacen uso de su API como DC-Unlocker y HCU-Client  podremos seguir desbloqueando el bootloader de nuestro dispositivo, pero no nos saldrá gratis ya que el uso de la API de HWdog es de pago.

Debido a esto os traemos un tutorial para poder desbloquear el bootloader de Huawei con estas herramientas ya que el tutorial que teníamos anteriormente ha quedado obsoleto, al menos hasta que Huawei cambie de opinión respecto a los códigos.

Obtener código para desbloquear bootloader Huawei

Antes de nada, tienes que revisar que tu dispositivo esté soportado actualmente, para ello tienes que visitar esta página. En caso de que no esté soportado, no garantizamos que éste método funcione y tendrás que esperar a que añadan soporte a ese dispositivo, aunque en el momento de redactar este tutorial el Huawei Mate SE no está soportado pero un usuario ha conseguido realizarlo sin problemas.

Instalar Drivers del Huawei

Para empezar, tenemos que tener los drivers de nuestro dispositivo instalados, en caso de no tenerlos, tendremos que descargar e instalar HiSuite.

Creando cuenta para obtener el código.

Después de eso tenemos que crearnos una cuenta para poder usar el programa HCUClient que será la que gestione nuestros créditos disponibles pinchando aquí. En el apartado de créditos tenemos que poner los créditos que queremos comprar según la operación que queramos realizar. Cada crédito cuesta 1€ y para obtener el código de desbloqueo de bootloader se necesitan 4 créditos. Puedes ver el precio de todas las operaciones disponibles aquí.

Una vez creada la cuenta y adquiridos los créditos, descargamos HCUClient desde aquí, descomprimimos el zip y ejecutamos HCU.exe. En caso de que al ejecutarlo nos de el error 0xc0150002, tendremos que instalar el paquete redistribuible de Microsoft Visual C++ 2008 de 32bits y 64 bits.

Utilización del programa para obtener el código de bootloader

Conectamos nuestro dispositivo mediante USB al PC, escribimos el siguiente código en la app de llamadas ##2846579## seleccionamos «Background settings» luego «USB ports settings» y «Manufacture mode«. Ahora en HCUClient iniciamos sesión con nuestra cuenta y seleccionamos «Read phone info» para leer los datos de nuestro dispositivo.

Una vez leídos los datos del dispositivo, vamos a la pestaña «Unlock» y seleccionamos «Read Bootloader Code«. Cuando seleccionemos esta opción, en el apartado de la derecha nos mostrará el progreso y finalmente el código de desbloqueo del bootloader de nuestro dispositivo.

Preparar teléfono para el desbloqueo

Una vez adquirido el código, solo queda desbloquear el bootloader de nuestro dispositivo. Hay que tener en cuenta que este proceso formateará nuestro dispositivo y en algunos modelos también borra la memoria interna, por lo que es recomendable hacer un backup de nuestros datos en una tarjeta SD o en el PC.

Antes de nada tenemos que descargar e instalar las herramientas ADB y Fastboot para poder realizar correctamente las operaciones (en la instalación seleccionamos todo Y).

También tendremos que desactivar el bloqueo FRP de nuestro dispositivo, para ello vamos a Ajustes>Información del sistema y pulsamos 10 veces sobre el número de compilación para desbloquear las opciones de desarrollador. Después de esto, vamos a Ajustes>Opciones de desarrollador y activamos la casilla «Habilitar desbloqueo OEM» y «Habilitar depuración USB«.

Una vez instalado, abrimos una ventana de comandos, para ello presionamos la tecla de Windows + R, escribimos cmd en el cuadro de búsqueda y presionamos enter.

Si tenemos el dispositivo conectado, escribimos lo siguiente para reiniciar en modo fastboot
adb reboot bootloader

Una vez en modo fastboot escribimos lo siguiente para ver el estado del bootloader
fastboot oem get-bootinfo

Desbloquear el bootloader Huawei con código

fastboot oem unlock xxxxxxx Donde xxxxxxx será nuestro código.

Una vez realizado todo esto nos saldrá una ventana para confirmar que queremos desbloquear el bootloader, seleccionamos «YES». Se nos reiniciará el dispositivo y ya tendremos el bootloader desbloqueado listo para hacer todo tipo de modificaciones.

Además de esto, HCUClient nos permite cambiar la región de nuestro dispositivo y eliminar el bloqueo FRP en caso de necesitarlo, pero al igual que el desbloqueo de bootloader también tendremos que pasar por caja.

Este tutorial no está patrocinado e InstaTecno no gana nada con ello, simplemente queremos informar de que aún existen procesos para desbloquear el bootloader de dispositivos Huawei y Honor aunque en este caso sea pagando.
